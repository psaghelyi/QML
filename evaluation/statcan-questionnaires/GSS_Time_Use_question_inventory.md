# General Social Survey, Cycle 2: Time Use - Question Inventory

## Document Overview
- **Title**: General Social Survey 1986 - Cycle 2: Time Use (GSS 2)
- **Organization**: Statistics Canada, Housing, Family and Social Statistics Division
- **Date**: July 1987 (survey conducted November-December 1986)
- **Pages**: 33 (including appendices)
- **Language**: English
- **Type**: CATI (Computer-Assisted Telephone Interview) questionnaire package with paper forms. Two versions: GSS 2-2 (full, 10,000 sample) and GSS 2-2A (without daily activities section, 6,500 additional sample for N.B., Quebec, Ontario)

## Structure
The questionnaire package contains:
- **Introduction**: Content, methodology, and sample description (pages 1-2)
- **Appendix A**: Main questionnaire GSS 2-2 with Sections A through S (pages 3-22, numbered 2-22 internally)
- **Appendix B**: Selection Control Form GSS 2-1 (pages 27-28)
- **Appendix C**: Extra Daily Activities Sheet GSS 2-2D (page 32-33)
- **Appendix D**: Code Sheet for Daily Activities (referenced but not included in PDF)

The main questionnaire (GSS 2-2) covers three major subjects:
1. **Social Mobility** (Sections A, B, P)
2. **Daily Activities / Time Use** (Section D)
3. **Well-being** (Section E)
4. **Language Knowledge and Use** (Sections F, G, H, J, K, L, M, N, Q, R/T)
5. **Background Characteristics** (Section S/U)

Note: Sections are routed based on language profile from Section F. Section T and Section U are alternate versions of Sections R and S used for respondents routed via Section T (English-only speakers). The GSS 2-2A omits Section D (Daily Activities).

### Section D: Daily Activities (Time Use Diary) - 59 question slots (each with 5 sub-questions)

[INTERVIEWER marks day of week: 1=Sunday, 2=Monday, 3=Tuesday, 4=Wednesday, 5=Thursday, 6=Friday, 7=Saturday]

The diary is structured as a sequential time-use log starting at 4:00 a.m. on the reference day. Each activity slot (1-44 on the main form, continuing from 45+ on extra form GSS 2-2D) contains the same 5 sub-questions:

**Activity slots 1-44 (main form) + additional slots on GSS 2-2D:**

Each slot N (for N = 1 to 44+) contains:

38. D_N_a: (First: Starting at 4:00 a.m., what were you doing? / Then: And then, what did you do next?) - **Open text**: activity code (2-digit)

39. D_N_b: When did this start? - **Numeric**: time (HH:MM) [Not asked for activity 1, which starts at 04:00]

40. D_N_c: When did this end? - **Numeric**: time (HH:MM)

41. D_N_d: Where were you? / Were you still... - **Radio** (Place): 1=R's Home, 2=Work Place, 3=Other Place; OR (In Transit): 4=Car, 5=Walk, 6=Bus & Subway, 7=Other

42. D_N_e: Who was with you? / Were you still... - **Checkbox**: 1=Alone, 2=Spouse/Partner, 3=Child(ren) of Household, 4=Other Family Member(s), 5=Friend(s), 6=Other Person(s) [Note: Do not ask question "e" about sleep, sex or other personal care activities]

**Unique sub-question nodes per activity slot: 5** (a, b, c, d, e)
**Total activity slots on main form: 44**
**Counting as unique question nodes: 5 sub-questions x 1 template = 5 unique question types, repeated up to 44+ times**

For inventory purposes, the daily activities section represents **5 unique question types** applied across a variable number of activity episodes (44 slots on main form, expandable via GSS 2-2D).

### Routing Notes
- Section F is a critical routing filter that determines which language sections (G/H/J/K/L/M) each respondent receives
- Only one of {G, H, J, K, L, M} is administered per respondent based on main language(s)
- Section N (childhood language) is administered to all respondents who pass through any language section
- Section T replaces Section R for English-only speakers (routed from F1 -> English -> No other language knowledge)
- Section U replaces Sections P+Q+S for GSS 2-2A respondents (additional sample without daily activities)
- Section D is omitted entirely from GSS 2-2A
